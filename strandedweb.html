Frequently Asked Questions

It's not working! Help!

This version of Stranded runs on RenPyWeb, support for individual browsers seems to be quite picky. If you see the following error message:

...then you can't run Stranded on this browser. Sorry.

If you're running the latest Chrome or Edge version, you can still play this game by changing the user agent to Firefox. (I recommend using a user agent changer extension. If you can't, then try using developer tools: F12 > ... > More tools > Network conditions > User agent: uncheck the 'use browser default' checkbox and select Firefox > F5 to refresh)

Despite best efforts, many corners had to be cut: the main menu video is now a static image, images appear blurry while assets are loading, music takes a while to play and sound effects may not play in time. To avoid these issues, I recommend that you download and play a local version of the game rather than in a web browser; this version is for players who have no local file saving or application running permissions.
